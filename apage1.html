Internet And Website Development

Introduction to the World Wide Web

How the web works

The world wide web is an information system on the internet, that is used while web browsing on a device. This system includes things like websites, files and all web-resources. We access the World Wide Web on a daily basis for all sorts of information via the internet. To use the world wide web, we don't need to know how to use coding systems, we just need to have the basic knowledge of what we are looking for and type it up on a web browser, shaping a pathway to the information we are looking for. The information is also accessible by entering a URL, a direct link to the website on the web or even a hyperlink. Think of it as a large server, where anyone can share information and retrieve information, using the internet.

The world wide web was invented in 1988 by Tim Berners Lee, an English scientist. The way to recognise if something is inside the web is if it starts with www. For example in the website www.google.com. we know that it is inside the world wide web as it starts with www.

Role of Web Servers

Web servers are a piece of software on every local computer. Web servers are servers where websites are uploaded. Web servers transfer raw materials (text, images, etc) and transfer to a web page, accessible by anyone accessing the world wide web. Whenever we type in a url in our web browser, it automatically gets transferred to a web server, which takes the request, processes it and sends it back to our computer where we can access the website. To publish a website you need to upload it onto a web server, a place where viewers can access it. Every large company has its own web server. For example, Microsoft has its own web server is called Internet Information Services (IIS).

Web Browsers

A web browser is an application used to access the world wide web. The browser transfers web page data from the web servers to the web page and vice-versa. Web browsers are means of accessing web pages. All operating systems come with their own version of web browsers. For e.g. Apple operating systems come with a default Safari browser. Microsoft uses Microsoft Edge as their default browser. While on most phones with Android systems, Chrome is the web browser used.

World Wide Web vs Internet

The Internet

The invention of the internet revolutionised the technological world and now, in the modern world, we are heavily dependant on the internet on a daily basis. The internet is a network that connects technological devices from all over the world. Through the internet, we can access the world wide web, as mentioned before where we can access and share web resources. The internet is not controlled by anyone and everyone has access to it.

The difference between the Internet and WWW

The internet and the world wide web are often said as the same, but they are not. The internet is the network connection of millions of computers worldwide, while the world wide web is the way we can access, share and receive information. To use the world wide web, we need the internet connection and without the internet, the world wide web would not operate. However the internet would still operate without the world wide web, it just would not be as useful. Think of the internet as a hotel, with elevators, stairs and pathways, where all the rooms and services are the web. The world wide web also uses the TCP protocol, whereas the internet can either use the same thing (TCP) or can also use the UDP protocol.

Intranet and Relation to Internet

What is an intranet?

An intranet is a private internet network that only authorised people can have access to. It can be used to share, access and receive web resources just like the internet. A common example of an intranet is the DOE portal, an education intranet. This private portal is only accessible to school students and staff and has restricted content and permissions. An intranet often keeps its content secure as only authorised members can access, and therefore is commonly used in companies, governmental organisations, and educational institutions.

Internet and Intranet Connections

The internet is a world wide network that connects all computers around the world. Documents, files and web resources can be viewed and shared by anyone and all computers can access. An intranet is also a network that connects computers and where people can share and view web resources. However, only authorised members have access to the intranet and only authorised members can access the information on their intranets.

Intranets generally have information that is classified and cannot be leaked, and to allow employees/members to access, they add it to a network where only they can see and edit. Internet Protocol (HTTP, TCP/IP and FTP)

Internet Protocol

Internet protocol is a certain set of rules or standards used to transfer data on the internet. There are many protocols but the most common two are HTTP and FTP, which both use TCP/IP as the underlying mechanism for data transfer.

TCP/IP

Tcp/IP is the inner-protocol that both HTTP and FTP use for data connections and management. The TCP part makes sure that all the data is in the correct order and that none of it is missing. It also maintains routing, to make sure the data is sent in the most efficient way possible in the circumstance the internet is slow. The IP part ensures that the routing is accurate. When the data transfer is in the form of a large file, TCP/IP makes sure that it is evenly distributed and sends it through the network.

HTTP

Http stands for hypertext transfer protocol and is the most efficient method of transferring smaller amounts of data. It is the protocol used by web pages. It is the method which web browsers use to communicate with web servers and underpins the communication of the world wide web. HTTPS is a secure http protocol where information sent is encrypted. Personal and financial data transfer should always be made over HTTPS as it is secure and encrypted. HTTP does not authenticate the connection and therefore can undertake multiple sessions simultaneously.

FTP

FTP stands for File transfer protocol and is the most efficient method of transferring large files of data over the network. FTP is only used to transfer data in a file-format and cannot be used for data transfer from web page to web server. FTP uses 2 TCP/IP connections, control connection and data connection. Control connection authenticates the user while data connection transmits data files. FTP can only maintain a limited number of sessions at a time. SFTP is secure FTP and is more secure as it transfers encrypted data.
